Git Flow

main (production) | |-- develop |-- feature/login |-- feature/dashboard |-- bugfix/crash

Common Git Commands.

# Setup git config --global user.name "Your Name" git config --global user.email "you@example.com" # Repo operations git clone git status git add . git commit -m "message" git push origin branch-name git pull # Branching git checkout -b feature/login git checkout main git merge feature/login git branch -d feature/login .

Basic Windows CMD Commands.

| Command | Description | | --------------- | ------------------------------------------------- | | `dir` | Lists files and folders in the current directory. | | `cd` | Changes directory (e.g., `cd Desktop`) | | `cd ..` | Moves one folder up | | `cls` | Clears the screen | | `exit` | Closes Command Prompt | | `echo Hello` | Prints text to screen | | `type file.txt` | Displays content of a text file | .

File and Folder Operations.

| Command | Description | | ----------------------------- | -------------------------------- | | `mkdir foldername` | Creates a new folder | | `rmdir foldername` | Deletes a folder (only if empty) | | `del file.txt` | Deletes a file | | `copy file1.txt file2.txt` | Copies a file | | `move file1.txt D:\NewFolder` | Moves a file | | `ren oldname.txt newname.txt` | Renames a file | .
